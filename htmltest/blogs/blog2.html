The Real, First Blog

12/31/24 - 01/02/25

I'm writing this on New Year's Eve, at 9:59pm. Crazy it's 2025 in 2 hours. I'll remember 2024 for a number of reasons, I had a few boxing fights, I learned how to play the guitar, and I learned HTML.

I've known HTML5 before 2024, but I only got around to some text and images, no CSS whatsoever. A couple weeks after that my Dad said he bought a web domain. (he's bought many, but this was the first domain he bought in a while) That got me back into coding and this time I went at it for more than 3 hours. Eventually, I had a very basic looking site.

What the site originally looked like. It had no CSS, so it looks like it's from the 90's.

An updated version of the site, this one has CSS in it, it's also the homestarrunner looking one.

I kept working on it, and it eventually turned into the site it looks like today. The site has gone through a few changes, (at one point it kind of looked liked one of homestarrunner.com's pages) but I like how it looks right now. I intentionally made it look like it is from the mid-2000's, which I think looks cool.

I have some ideas and plans for this site, like different browser themes, (a hacker-themed one or a holiday-themed one for example) adding more things like forums, and more.

I really turned this into a blog lol. I was just trying to fill out the homepage, but this works!
