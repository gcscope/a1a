Thank You Claude

06/26/2026

2-3 days ago, I tried using Claude. I've known about it and wanted to try it for a while, but I always forgot and stuck to ChatGPT. But I remembered and decided to try it out.

I've heard it's really good for coding, so I asked it to make a complex website with a dark tone, and it looked really good. I don't have an image of it, since it's on another computer, so I remade it using Claude and it came out looking like this:

It even found images on the web to use for examples, Which is pretty cool in my opinion.
So after I saw an example of what Claude can really do, I went onto my Mac, uploaded my website files to Claude, and told it to rate my site and tell me what changes I could make. Claude told me that I should add a dark mode. That made me curious, so I asked it to update the code to add a dark mode switch, and it looked better than what I thought.
It obviously didn't come out perfect on the first try (The header would become white on light mode instead of staying the same colour), so I asked it to make some changes, and it eventually became the dark mode I have right now.
I also told it to edit the buttons, since what they looked like on Firefox didn't match how they looked like on Safari, or on my iPhone (I set up a localhost server using Python). I edited the font, border and colour so it looked good across all web browsers.
I know my original goal making this webiste was to make it from scratch using only TextEdit, and using Claude is kind of breaking that promise, but the only things Claude has done for my site were the things I listed above. Everything else I made from scratch. If I put it into a percentage, I'd say 85% of the site was made by me, and 15% was made by Claude.

Overall, I think trying Claude was a good thing, because its given me motivation to work on this website again. I asked it how to setup a github page, and if you could conect custom domains to it (I honestly didn't know) and Claude told me how to (even though I could've done a quick Google search). So Thank You Claude.
